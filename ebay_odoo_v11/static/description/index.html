Setup and Configure

After installation of this module and once users gets e-commerce manager acces we will be able to see menu for "Market-Places" and here we can create instances for our eBay shop.

We can fill eBay credenctials for our eBay Instance.

After creating Instance a Shop related to that particular Instance is getting created. We can clearly see Shops getting creating in this image.

In this shop we will see four tab in these tabs eBay action button defined for various functions.

This tab has eBay store information. by hitting on "Import Store" button we can fetch all the store information in odoo with custom category as we can see here in this image.

Import eBay Attribute set

We can go to Mareket-places --> Attribute --> Attribute set and We can import attribute set of any particular attribute.

We can put attribute name,code,eBay shop and hit on "Get item/condition enabled" and "Get Category Specifics" buttons to get attribute information from eBay shop.

Import eBay Attribute

We can see imported attribute of eBay shop.

Import eBay Attribute Options

We can see imported attribute options of eBay shop .

Create eBay Product Template

We can create eBay product template in Mareket-places --> eBay listing --> eBay Template. In first tab we can fill product listing infomation for eg. Template type ,Template name,Currency,Price,Best Offer and Description

We can fill Buyer Requirements,Unpaid Strike,Violation Policy,Feedback,Private Listing.

We can fill return policy.

We can fill the shipping type.eg.- Handling time,Additional option,Domestic/International Shipping.

Create eBay Site

We have created few default sites. We can create multiple sites which will be used in while creating instances.

eBay Product Listing

We can create all type of listing for example single and variation with Listing Duration (GTC,1 days,3 days,30days etc..) variation and single listings is explained below.

eBay Variation Product Listing

Once we checked "Variation Product" check box then we will be able to see "Variation Details" and we fill variation product information eg. - Variation Subtitle,Variation Description,Image.

We can upload image of parent product.

Add child products of parent.

Add child products attribute.

We can add eBay category1 ,store and attribute information.

We can add eBay category2 ,store and attribute information. eBay allow us to upload products in more then one category. If all information updated correct than click on "Add Product In Ebay". If we make any changes in future and we want to update that changed information on eBay we can achieve this by "Update Products In Ebay" button.

Once product is listed on eBay Online Shop than Itemid and listing Id gets automatically updated in odoo and It also gets updated in products eBay listing.

Ebay product attribute--> This is product information page with all details under tab "Ebay Information".

eBay Single Product Listing

We can upload Single Product without any variation on eBay online Shop.

We can add single product information.

We can add eBay category1 ,store and attribute information.

We can add eBay category2 ,store and attribute information. eBay allow us to upload products in more then one category. If all information updated correct than click on "Add Product In Ebay". If we make any changes in future and we want to update that changed information on eBay we can achieve this by "Update Products In Ebay" button.

After hitting on button "Add Product In Ebay" if product listed on eBay than Itemid and listing id gets automatically updated in odoo and it also updated in products eBay listing.

Import eBay Carrier

From eBay shop we can import eBay carrier.

We will see all imported carrier.

Import eBay Order WorkFlow

This is setup for order workflow. in this we can set order policy. eg. - If we want paid orders to be imported then we can set this in workflow.

Import eBay Order

From eBay shop we can import eBay order.

As per configuration set in workflow confirmed orders are coming in odoo.

Update eBay Order Status

From eBay shop we can update order status.

We can also update order status from order. select order which we want to update and click on action select "update order status".

We can see all selected order in wizard and we can update tracking no and carrier by hitting on "Update Order Status" button.

Update eBay Products Stock And Price

From shop you will update product stock and price by clicking on "Export Price" and "Export Stock" buttons.

We can also update Stock and Price in selected products .

We have facility to select any eBay shop with which we want to update price--> hit on button "Update Price", "Update Stock" and we can also update both on single button hit.

eBay Automation

We can create cron job(schedular) which will on regular interval configured in odoo so that Importing of Orders, Updating orders Status, updating stock and price etc....
